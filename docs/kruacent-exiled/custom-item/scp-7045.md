## 1. Rapport de Neutralisation

**SCP-7045** est un enregistreur de microcassettes de marque █████, modèle 2001.
Dans son état actif, l'objet possédait la capacité anormale de se téléporter de manière autonome pour enregistrer des événements historiques majeurs, souvent liés à la fin de mondes parallèles.

**L'Expérience du Site-19 :**
Le **Dr. Patrique** a tenté de "recalibrer" l'objet pour l'utiliser comme système de messagerie interne sécurisé pour les agents de terrain.
Pendant une courte période, l'objet a fonctionné : il permettait à un agent d'enregistrer un message vocal, puis se téléportait instantanément dans la poche d'un autre agent pour délivrer l'information.

Cependant, le **Dr. Patrique** a noté une dégradation progressive du signal. Les voix devenaient robotiques, saturées par le "bruit de fond" de l'univers qui s'effondre (ce que les techniciens appellent vulgairement *Codecs Audio Pourris*).
Finalement, après avoir tenté d'enregistrer le son d'une explosion nucléaire, l'objet a émis un dernier *CLIC* et a cessé toute activité anormale. Il est aujourd'hui considéré comme **NEUTRALISÉ**.

---

## 2. Ancienne Procédure

:::danger OBJET DÉSACTIVÉ
Cet objet n'apparaît plus dans le jeu ou est totalement inerte. Cette section est conservée à titre d'archive historique sur les capacités passées de l'anomalie.
:::

Lorsque l'objet était actif, il fonctionnait selon le cycle suivant :

| Phase | Action | Résultat |
| :--- | :--- | :--- |
| **1. Record** | Utilisation en main | L'utilisateur parlait dans le microphone. |
| **2. Teleport** | Fin de l'enregistrement | L'objet se dématérialisait instantanément. |
| **3. Playback** | Matérialisation | L'objet réapparaissait aux pieds d'un joueur aléatoire (souvent dans une autre zone) et jouait le message. |

## 3. Archives Récupérées : "Victor & Harry"

Avant sa panne définitive, SCP-7045 contenait une cassette inamovible relatant les derniers instants de deux hommes, Victor et Harry, errant dans une réalité où la Fondation a échoué.

> **[CLIC - LECTURE]**
>
> **Victor :** "Le ciel s'est déchiré en deux. Les biomes se sont inversés... Je m'appelle Victor. Je travaille pour la Fondation SCP. Nous étions censés contenir les monstres, mais je crois que l'un d'eux s'est échappé."
>
> **Harry :** "Tu parles encore à ce truc ? Victor, regarde ton pied... l'os est visible."
>
> **Victor :** "Je ne sens plus rien, Harry. La réalité s'effrite. Je veux juste laisser une trace. Si quelqu'un trouve ça... sachez que nous avons essayé."
>
> *[Bruit de vent hurlant et de statique intense]*
>
> **Victor :** "Je me demande si je vais revoir ma femme... Non, je ne veux pas la voir mourir. Harry ?"
>
> **Harry :** *[Silence]*
>
> **Victor :** "Harry a disparu. Il a juste... cessé d'exister. C'est mon tour. Adieu."
>
> **[CLIC - FIN DE BANDE]**
